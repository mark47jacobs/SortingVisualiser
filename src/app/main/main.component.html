Sorting Visualiser

This tool is for noobs,who need a visual for understanding Sorting Algorithms . It visualises various sorting algorithms like QuickSort,MergeSort,SelectionSort and BubbleSort .

Correct way to use is 1. select sorting speed 2. then click on Set Speed & Generate New Array 3. then select a sorting algorithm 4. then click on sort now.

Warnings!!!

Do not change the sorting speed while sorting is cgoing on!!!

Do not change the sorting algorithm and/or use the sort now button while sorting is going on!!!
